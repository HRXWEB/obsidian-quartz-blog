Inspections 默认情况下会自动运行，边写代码就会边分析。

# 手动运行

也可以手动运行 Inspections，有几种可选的方式和相应的配置：

1. 在主菜单，**Code | Inspect Code**
    1. 可以选择分析范围
    2. 可以选择检查的配置，即检查规则
2. 在主菜单，**Code | Analyze Code | Run Inspection by Name**
    1. 通过名字选择需要检查的某些类型的问题
    2. 选择分析范围

# 在代码提交到Git之前运行

如下图所示：

1. 在 commit tool window 点击 设置的小齿轮icon
2. 勾选 `Analyze Code`
3. 点击Configure 选择分析的规则配置
